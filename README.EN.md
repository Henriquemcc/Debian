# Workstation with Debian, Ubuntu or Linux Mint

This is my installation script for the programs I use on my computer in my Linux distribution.

Supported Linux distributions:

[Debian](https://www.debian.org)

[Ubuntu](https://ubuntu.com)

[Linux Mint](https://www.linuxmint.com)

## How to run

With Debian, Ubuntu or Linux Mint installed on your machine, open the 'Terminal' and on it copy and paste the following command:

```
curl -L -O "https://github.com/Henriquemcc/Debian/raw/refs/heads/main/Iniciar.bash" && bash ./Iniciar.bash
```
