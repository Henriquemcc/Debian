# Workstation com Debian, Ubuntu ou Linux Mint

Este é o meu script de instalação dos programas que utilizo no meu computador na minha distribuição Linux.

Distribuições Linux suportadas: 

[Debian](https://www.debian.org)

[Ubuntu](https://ubuntu.com)

[Linux Mint](https://www.linuxmint.com)

## Como executar

Com o Debian, Ubuntu ou Linux Mint estando instalado em seu computador, abra o programa 'Terminal' e nele copie e cole o seguinte comando:

```
curl -L -O "https://github.com/Henriquemcc/Debian/raw/refs/heads/main/Iniciar.bash" && bash ./Iniciar.bash
```
